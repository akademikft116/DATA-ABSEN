Panduan Penggunaan

Langkah 1: Upload File Excel

Upload file Excel dengan format data presensi. Pastikan file berisi kolom: Nama, Tanggal, Jam Masuk, Jam Keluar.

Langkah 2: Konfigurasi Perhitungan

Atur parameter perhitungan gaji (opsional). Jika dikosongkan, sistem akan menggunakan nilai default.

Langkah 3: Proses Data

Klik tombol "Proses Data" untuk memulai kalkulasi otomatis.

Langkah 4: Download Laporan

Download hasil pengolahan dalam format Excel yang sudah lengkap dengan perhitungan keuangan.
