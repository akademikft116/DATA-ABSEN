Petunjuk Penggunaan

  1. Upload file Excel data presensi dengan format kolom: Nama, NIK, Tanggal, Jam Masuk, Jam Keluar
  2. Atur opsi pengolahan data sesuai dengan kebijakan universitas
  3. Klik tombol "Proses Data Presensi" untuk memulai perhitungan
  4. Setelah proses selesai, preview data akan muncul di bawah
  5. Unduh file hasil pengolahan dalam format Excel yang sudah termasuk perhitungan keuangan
  6. File yang diunduh ada dua: data asli presensi dan data yang sudah dihitung dengan template keuangan
